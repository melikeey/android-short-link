# SHORTLY

Android application that converts long links into short links

## Technologies
 **Kotlin Language, MVVM Architecture**

 - Material Design 
 - Lifecycle
 - Navigation
 - Coroutines 
  - Hilt
  -  Retrofit 
 - Room
